# Cadastro de Médicos

Trata-se de um pequeno sistema web de demonstração escrito na linguagem Java, no qual é possível gerenciar um cadastro de médicos e especialidades.

- Plataforma: Java EE 8
- IDE: Eclipse 2018-09
- Tipo de projeto: Maven Java Web Application
- Frameworks: JSF 2, Omnifaces, Hibernate 5, Apache Commons Validator, JQuery, Bootstrap 4
- Banco de dados: PostgreSQL 11
- Servidor: WildFly 14
